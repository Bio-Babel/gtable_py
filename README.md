# gtable_py

Python port of the R **gtable** package.

## Installation

```bash
pip install -e ".[dev]"
```

## Quick Start

```python
import gtable_py
```

## R Version Mapping

| Python | R |
|---|---|
| 0.1.0 | (initial port) |

## Documentation

```bash
pip install -e ".[docs]"
mkdocs serve
```
